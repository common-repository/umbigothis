Social bookmarking plugin for Wordpress.

== Description ==

UmbigoThis é um plugin de compartilhamento social, que possibilita que
seus visitantes adicionem seus posts em diversas redes de compartilhamento
de links.

Ao ser instalado, ele adiciona 25 links para redes sociais automaticamente, evitando que
você tenha que criar os links manualmente.

A partir da versão 0.4 ele ganhou dois modos:
1. Normal: exibe a janela popup com a lista dos links de compartilhamento.
2. Modo ícones: exibe os ícones das redes sociais logo abaixo do post.

É utilizado jQuery para criar os efeitos e a janela popup.

== Installation ==

Esta seção explica como instalar o plugin:

1. Descompacte o arquivo 'umbigothis.zip'
2. Faça upload dos arquivos para o diretório 'wp-content/plugins' da sua instalação do WP
3. Acesse o painel de controle do Wordpress e ative o plugin
4. Pronto!
